Qu'est-ce qu'un point de fuite ?


Un point de fuite est le point d'intersection de deux lignes parallèles

    • Si la ligne la plus proche de cette voiture sur le côté droit est sélectionnée

      Si la ligne la plus proche de cette voiture sur le côté gauche est sélectionnée


Dans cet exemple, Nous avons trouver le point de fuite qui est le point vert
